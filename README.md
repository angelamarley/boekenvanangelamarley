# Boeken van Angela Marley

Upload alle bestanden en mappen uit deze zip naar de GitHub-repository `boekenvanangelamarley`.

Daarna:
1. Open **Settings**
2. Kies links **Pages**
3. Kies **Deploy from a branch**
4. Selecteer **main** en **/(root)**
5. Klik **Save**

De website komt daarna op:
https://angelamarley.github.io/boekenvanangelamarley/
